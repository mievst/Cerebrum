# Воркеры проекта Cerebrum

## Общая архитектура

Все воркеры интегрированы с единой системой очередей Celery и используют Redis для хранения результатов. Каждый воркер может обрабатывать определенный тип задач.

## Python воркер (math)

Обрабатывает математические задачи, удваивая значение.

### Использование:
```json
{
  "type": "math",
  "value": 5.0
}
```

## Video воркер

Обрабатывает видео задачи, вычисляя длительность видео и создавая обработанную версию.

### Использование:
```json
{
  "type": "video",
  "video": "/path/to/video.mp4"
}
```

## C# воркер

Обрабатывает задачи по преобразованию текста в верхний регистр через Python мост.

### Использование:
```json
{
  "type": "csharp",
  "text": "hello world"
}
```

## Rust воркер

Обрабатывает математические задачи с различными операциями через Python мост.

### Поддерживаемые операции:
- `multiply` - умножение
- `add` - сложение
- `power` - возведение в степень
- `sqrt` - квадратный корень
- По умолчанию - удвоение значения

### Использование:
```json
{
  "type": "rust_math",
  "value": 5.0,
  "operation": "multiply",
  "multiplier": 3.0
}
