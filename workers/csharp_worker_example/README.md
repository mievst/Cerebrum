# C# Worker с Celery Bridge

## Описание
Этот воркер написан на C# и использует мост для взаимодействия с Celery.
Поскольку C# не имеет встроенной поддержки Celery, мы используем промежуточный
Python скрипт для моста между Celery и C# приложением.

## Архитектура
- Celery воркер на Python (celery_bridge.py) получает задачи из очереди
- Python мост вызывает C# приложение через subprocess
- C# приложение обрабатывает задачу и возвращает результат
- Python мост отправляет результат обратно в Celery

## Зависимости
- Python 3.8+
- Celery
- redis
- .NET 8.0

## Установка и запуск

1. Сборка C# приложения:
```bash
dotnet build
```

2. Запуск Celery воркера:
```bash
celery -A celery_bridge worker --loglevel=info
```

## Как это работает
1. Celery воркер получает задачу из очереди
2. Вызывается функция process_csharp_task
3. Python subprocess вызывает C# приложение с параметрами задачи
4. C# приложение обрабатывает задачу и возвращает результат в stdout
5. Python мост парсит результат и возвращает его в Celery
